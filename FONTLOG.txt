FONTLOG for Zad
================

Zad is the Gulf Type Foundry's UI workhorse — a sans-serif designed for
dense, operational interfaces in both Arabic and Latin. It anchors the
foundry's own studio and website.

ChangeLog
---------

1. 2026-06-12 — Direction-C identity pass
   Zad becomes the foundry's default UI face. Tightened metrics for dense
   tables, refined Arabic letterforms, locked the weight ladder
   (Regular 400, Medium 500, Bold 700).

2. 2026-06-04 — Initial release of Zad
   First public version. Arabic + Latin coverage. Released under SIL
   Open Font License 1.1.
